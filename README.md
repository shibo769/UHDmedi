

# UHDmedi

**UHDmedi** is an R implementation for estimating total, indirect, and direct effects under ultra-high dimensional setting (UHD) via debiasing, including potential treatment-covariates, treatment-mediators interaction effects. This implements methods described in:

Bo, S., Ghassami, A., & Mukherjee, D. (2025). "A Debiased Estimator for the Mediation Functional in Ultra-High-Dimensional Setting in the Presence of Interaction Effects."[arXiv:2412.08827](https://arxiv.org/abs/2412.08827)
